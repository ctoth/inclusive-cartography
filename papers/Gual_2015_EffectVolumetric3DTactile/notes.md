# The effect of volumetric (3D) tactile symbols within inclusive tactile maps

## One-Sentence Summary
An empirical, within-participants Spanish field study showing that adding 3D-printed volumetric point symbols on top of a conventional microcapsule (swell-paper) tactile map substantially reduces location time and discrimination errors for blind and visually-impaired users — especially blind users — compared with standard "flat" microcapsule-only maps. *(p.1, p.8, p.9)*

## Problem Addressed
Conventional tactile map production (silkscreen, embossing, thermoform, microcapsule/swell paper) yields only quasi-2D symbols with 0.5–1 mm relief. Because point symbols live on the same tactile plane as area hatchings, lines, and background textures, blind and visually impaired users struggle to discriminate point symbols from surrounding map content and to discriminate symbols of similar shape from each other. Prior work proposed volumetric (3D) symbols but few studies compared 3D vs flat inclusive maps under matched conditions. *(p.1, p.2)*

## Key Contributions
- Introduces a "mixed" inclusive tactile map: a microcapsule/swell-paper base (lines, areas, hatchings, Braille legend) combined with FDM 3D-printed ABS volumetric point symbols glued on top. *(p.2, p.4)*
- Reports a controlled counterbalanced study (N=16, three user profiles: blind/BVI, visually impaired, sighted blindfolded) comparing a flat map (Map A) vs a mixed 3D-enhanced map (Map B) on three tasks: symbol identification without context, symbol identification on-map without legend, and targeted location using legend. *(p.3–p.5)*
- Provides quantitative evidence that volumetric symbols reduce mean location time and mean error counts for all three user profiles, with the strongest gains for blind users, and demonstrates a learning-transfer advantage for the group that started with Map B (3D first). *(p.5–p.8)*
- Argues that 3D symbols close the gap between blind and visually-impaired users and raise autonomy, supporting "inclusive" tactile cartography where one artifact serves multiple visual profiles. *(p.8–p.9)*

## Study Design
- **Type:** Between-subjects (two counterbalanced groups) quantitative user study with repeated measures within each group across Maps A and B. *(p.3)*
- **Population:** N=16 participants, ages 14–74, recruited via ONCE (Spanish national org for the blind) and the Bromera Cultural Centre in Alzira (Valencia). Three visual profiles: totally blind, visually impaired (low vision / late-blind), and sighted participants blindfolded as a control. Each of the two groups (G1, G2) contained 8 participants distributed across profiles. *(p.3, Table 1)*
- **Intervention(s):**
  - **Map A (flat):** Microencapsulated swell-paper map only. Point symbols are the standard flat shapes produced by the microcapsule process (heated on a Zychem fuser), with ~0.5–1 mm relief. *(p.1, p.4)*
  - **Map B (mixed / 3D):** Same microencapsulated base map as Map A but with 10 FDM-printed ABS volumetric point symbols (3–8 mm tall) adhered on the designated symbol positions. *(p.2, p.4)*
- **Comparator:** Map A (flat) served as the comparator/baseline within the counterbalanced design. *(p.3)*
- **Counterbalancing:** G1 performed the tasks on Map A first, then Map B. G2 performed on Map B first, then Map A. This supports assessment of both an absolute Map A vs Map B effect and a learning/order effect. *(p.3, p.5)*
- **Primary endpoints:**
  - Time to locate / identify target symbol(s) (seconds). *(p.5, Tables 3–5)*
  - Number of discrimination / location errors per task. *(p.5, Tables 6–9)*
- **Secondary endpoints:** Subjective assessment along three axes: effectiveness, efficiency, autonomy (informally reported in the discussion). *(p.4, p.8)*
- **Follow-up:** Single-session study; no longitudinal follow-up. *(p.4)*

## Methodology
- **Setting:** Quiet room at a community/cultural centre; participant seated at a table with the tactile map in front of them; Braille legend printed on a side sheet. Researchers timed each task and scored errors. Participants were told the map represented a real building layout (the Bromera Cultural Centre / public building). *(p.4, Fig 8)*
- **Training phase:** Before the tasks, each participant received a short symbology training session in which they were shown/taught each of the 10 point symbols (flat and 3D variants) on an isolated training card, to establish a baseline of familiarity. *(p.4)*
- **Tasks (each performed on both Map A and Map B within a group):**
  1. **Task 1 — Symbol identification (context-free):** Given each of the 10 point symbols isolated, name it. Measures shape discrimination of symbols alone. *(p.4–p.5, Table 3)*
  2. **Task 2 — Location without legend:** Find specified elements on the map using only previously-taught symbol knowledge. Measures retention and in-context discrimination. *(p.4–p.5)*
  3. **Task 3 — Location with legend:** Find specified elements using the Braille legend as a reference. Measures functional map-use efficiency. *(p.4–p.5)*
- **Measured variables per task:**
  - Location time in seconds.
  - Number of discrimination / location errors (counted and classified by type: symbol confusion, off-target location, failure to locate).
- **Statistical analysis:** Non-parametric tests given the small sample size. Mann-Whitney U test used to compare groups on error counts and completion times. Reported as means (M) and standard deviations (SD), with group-level N, in Tables 3–15. Significance threshold α = 0.05 (standard convention). *(p.5–p.7)*
- **Production methods:**
  - Swell paper (microcapsule): A4 sheet printed on a laser printer, then heated in a Zychem / Minolta fuser; heat swells the black ink regions into a raised relief of ~0.5–1 mm. *(p.1, p.4)*
  - 3D volumetric symbols: Designed in CAD, fabricated by fused deposition modelling (FDM) in ABS plastic; one symbol per map feature (10 symbols total); attached to the microcapsule map with adhesive. Volumetric heights in the 3–8 mm range (cones, pyramids, truncated volumes, round-top forms). *(p.2, p.4)*

## Key Equations / Statistical Models
No closed-form mathematical equations or regression model specifications are given in the paper. The analysis is descriptive (M, SD) plus non-parametric group comparisons (Mann-Whitney U). *(p.5–p.7)*

The comparison model used per metric can be written schematically as:

$$
U = n_1 n_2 + \frac{n_1(n_1+1)}{2} - R_1
$$

Where: $n_1, n_2$ are the sample sizes of the two groups being compared (G1, G2), and $R_1$ is the rank sum of group 1. This is the standard Mann-Whitney U form implied by the paper's reference to "Mann-Whitney U" as the non-parametric comparison test. *(p.5)*

## Methods & Implementation Details
- **Map design protocol:** Both Map A and Map B depict the same floor plan of the Bromera Cultural Centre (Alzira, Valencia). The microcapsule base is identical across both versions, so that any difference in outcomes is attributable to the addition of 3D symbols on Map B. *(p.4)*
- **Symbol set:** 10 point symbols with geometric primitives (circle, square, triangle, cross, plus pictorial forms for stairs and kitchen stove). Map A uses the flat (swell) versions; Map B uses the same 10 shapes re-expressed as volumetric solids. *(p.2, Fig 3)*
- **Disambiguation rationale:** Flat swell versions of a circle and a small filled dot (or circle and ring) are easily confused under finger exploration; volumetric versions differentiate by profile (e.g. cone vs truncated cylinder vs dome), adding a "height-and-shape" channel that subjects exploit by sliding fingertips along the top. *(p.8)*
- **Training:** Short structured familiarisation of each participant with the 10 symbols before measurement begins. *(p.4)*
- **Task administration:** Fixed task order per participant (Task 1 → Task 2 → Task 3) within a map session. Counterbalancing is between groups (Map A vs Map B first), not within. *(p.3–p.5)*
- **Timing:** Researchers timed each task with a stopwatch from the start of exploration to the participant's verbal answer/identification or self-declared completion. *(p.4–p.5)*
- **Error scoring:** Errors counted per task; classified as (a) symbol misidentification, (b) incorrect location on map, (c) non-response after threshold. *(p.5)*
- **Statistical workflow:** Means and SDs computed by group and profile, then compared with Mann-Whitney U (small N, non-normal distributions). *(p.5–p.7)*

## Results Summary
- For Task 1 (symbol identification without map context), mean time per symbol and mean errors are lower on Map B (3D) than Map A (flat) for the blind and visually-impaired profiles; for sighted-blindfolded participants the difference is smaller but in the same direction. *(p.5, Tables 3–5)*
- For Task 2 (location without legend) and Task 3 (location with legend), Map B (3D) outperforms Map A (flat) on both mean time and mean error counts across all profiles, with the largest effect in the blind subgroup. *(p.5–p.7, Tables 6–9)*
- Group 2 (which started with Map B) shows a clear transfer-learning advantage when subsequently using Map A, suggesting that early exposure to 3D volumetric symbols helps users build a robust mental model of the map that transfers to flat maps. *(p.5–p.6)*
- Relative gains, as discussed qualitatively in §4: ~20–30% faster location for blind users on Map B in Case 1 (without legend) and ~30–50% faster on Map B in Case 2 (with legend); error counts for visually-impaired users drop from ~5 to ~1.5 per task. *(p.8)*
- Performance gap between visual profiles narrows on Map B: sighted-blindfolded, visually impaired, and blind participants converge on the 3D map. *(p.8–p.9)*
- The paper treats Mann-Whitney U p-values as the gate for "statistically significant"; several comparisons are reported as statistically significant, others as directional but non-significant given N=16. *(p.5–p.7)*

## Limitations
- Small sample (N=16 total, ~5 per profile per group). Statistical power is limited; non-parametric tests mitigate distributional assumptions but not power. *(p.3, p.9)*
- Single-session design; learning and fatigue effects partly controlled by counterbalancing, but no longitudinal follow-up. *(p.4)*
- Only 10 point symbols tested; does not cover line and area volumetric symbols (authors mention these as future work). *(p.9)*
- Map depicts a single building layout; generalisation to outdoor / geographic / larger-area tactile maps is not directly tested. *(p.4, p.9)*
- FDM-printed ABS volumetric symbols are manually glued onto swell paper; the authors note this as a production overhead and do not benchmark throughput vs pure microcapsule production. *(p.4)*
- Subjective outcomes (effectiveness, efficiency, autonomy) reported narratively, not quantitatively. *(p.4, p.8)*
- No reporting of inter-rater reliability for error classification. *(p.5)*

## Arguments Against Prior Work
- **Against flat tactile production technologies as sufficient:** Silkscreen, embossing, thermoform, and especially microcapsule/swell paper all produce symbols in a tight 0.5–1 mm relief band that is indistinguishable in height from background linework and area hatchings, so point symbols cannot be discriminated by height. *(p.1)*
- **Against symbol sets based on 2D shape primitives alone:** Similar flat shapes (circle vs dot vs ring; square vs diamond) confuse blind users during finger-sliding exploration — the paper demonstrates empirically that on Map A confusion rates are high for these pairs, but on Map B the volumetric profile disambiguates them. *(p.2, p.8)*
- **Against treating blind and visually-impaired users as a single homogeneous population:** Prior evaluations often lump profiles together. The paper shows that flat maps produce a wide profile gap but 3D-enhanced maps close it, implying that non-stratified evaluations may underestimate 3D benefits. *(p.8–p.9)*
- **Against dismissing volumetric symbols as a novelty:** The authors argue that FDM is now accessible enough that a small batch of volumetric point symbols can be added to any swell-paper base, keeping the inclusive map compatible with the established microcapsule production pipeline rather than replacing it. *(p.2, p.4, p.9)*

## Design Rationale
- **Hybrid microcapsule + 3D-printed stack:** Retains the fast, low-cost, high-resolution line and area rendering of swell paper for routes, walls, boundaries, and Braille labels, while adding volumetric discriminability only where it matters — at the point symbols. Alternative approaches (fully thermoformed 3D maps; CNC-milled maps) are rejected as too slow, too expensive, or too low-resolution for lines and Braille. *(p.1, p.2, p.4)*
- **Choice of FDM/ABS for volumetric symbols:** FDM is cheap, accessible to small labs and community producers (ONCE, schools), and produces symbols with enough height difference (3–8 mm) to be clearly above background relief. Alternatives (SLA resin, photopolymer) are not chosen because the authors prioritise reproducibility on accessible hardware. *(p.2, p.4)*
- **Counterbalanced two-group design:** Chosen to disentangle absolute map effects from learning/order effects, so that the 3D advantage is not confounded by simple task familiarity. *(p.3)*
- **10-symbol set sized to a single building:** A realistic scale for indoor wayfinding and small enough to finish tasks in one session per participant. *(p.4)*
- **Inclusion of sighted-blindfolded participants:** Serves as a control to distinguish "tactile-novice" effects (sighted blindfolded struggle more than blind experts) from "symbol-discriminability" effects (which should affect all profiles). *(p.3, p.9)*

## Testable Properties
- For any participant profile, mean location time on a 3D-enhanced map must be strictly lower than on the equivalent flat-only map for matched tasks. *(p.5–p.8)*
- Mean error count on the 3D map must be lower than on the flat map for blind participants, and approximately halved in the legend-assisted location task. *(p.8)*
- Participants who first use the 3D map must perform at least as well on the subsequent flat map as participants who first use the flat map perform on the subsequent 3D map (asymmetric transfer hypothesis). *(p.5–p.6)*
- The performance gap between "blind", "visually impaired", and "sighted blindfolded" profiles must narrow when moving from flat to 3D maps. *(p.8–p.9)*
- Volumetric symbol heights in the 3–8 mm range must be discriminable from a swell-paper base of 0.5–1 mm relief by finger exploration alone. *(p.2, p.4)*
- Training on 10 point symbols must be sufficient for baseline discrimination within a single session for participants aged 14–74 across all three profiles. *(p.3–p.4)*
- Differences between Map A and Map B on error counts must yield p < 0.05 under Mann-Whitney U for at least the blind subgroup. *(p.5–p.7)*

## Relevance to Project
This paper is directly relevant to the inclusive-cartography project's Seed §3.3 scoping exercise: it is a primary empirical data point for the claim that 3D volumetric enhancements over a microcapsule base (a hybrid, "inclusive" stack) materially improve BVI wayfinding performance vs flat-only maps. For the systematic review's LLM-confidence-threshold critique, it is a useful test case because (a) the paper's core claims are operationalised quantitatively (time, errors, Mann-Whitney U), (b) its sample is small and non-parametric — so any automated claim extractor should flag it as "small-N evidence" rather than strong; (c) it explicitly reports direction-of-effect and magnitude, not just significance, which lets reviewers test whether the LLM preserves effect size as a first-class claim.

For implementation (if the project ever prototypes inclusive map production), the paper also provides concrete design parameters: A4 swell base, 0.5–1 mm relief, 10 FDM-printed volumetric point symbols at 3–8 mm height, glued onto the microcapsule base, with Braille side-legend.

Layering both yields a multiplicative design rule: 3D symbols matter (Gual) AND audio-tactile beats verbal-only (Papadopoulos).
- [[Brock_2015_InteractiveMapsUsability]] — Moderate. Same-year (2015) empirical study of tactile map accessibility but different interaction paradigm (audio-tactile on flat vs volumetric tactile). Complementary channels that could be combined in an inclusive map design.
- [[Perkins_2002_CartographyProgressTactileMapping]] — Moderate. Perkins 2002 is the canonical literature review that defines "tactile cartography" as a research programme; Gual 2015 is a direct contribution within that programme, proposing volumetric point symbols as the missing fourth element class beyond points/lines/areas.
- [[Taylor_2016_Customizable3DPrintedTactile]] — Moderate. Taylor 2016 provides a customisable 3D-printed tactile map production pipeline (web interface → print). Gual 2015 is one of the empirical justifications for adding 3D relief to tactile maps at all; Taylor is the tooling side of the same design hypothesis.
- [[Götzelmann_2016_LucentMaps3DPrintedAudiovisual]] — Moderate. Götzelmann's LucentMaps uses 3D-printed volumetric tactile structures with embedded light/audio channels; Gual's volumetric symbols are the tactile-only ancestor of the same concept.
- [[Wabinski_2022_GuidelinesStandardizingTactileMaps]] — Moderate. Wabinski's standardisation guidelines for tactile maps need to decide whether volumetric point symbols belong in a standard, and Gual 2015 is the primary small-N empirical data point bearing on that decision.
